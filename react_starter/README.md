# Interactive Portfolio Builder Workshop

Welcome to the Portfolio Builder workshop! In this hands-on session, you'll build a professional portfolio website using React, learning key concepts about data-driven UI development.

## What You'll Learn

- Component architecture and reusability
- Props and data flow in React
- Dynamic rendering with `.map()`
- Data-driven UI development
- Responsive design with Tailwind CSS
- Deployment to production
  - We will focus on deploying to Vercel for this presentation
  - Github Pages and Netlify are also great options

## Quick Start

### Prerequisites

- Node.js installed
- Basic HTML, CSS, and JavaScript knowledge with some familiarity with React is helpful
- A code editor (VS Code recommended)

### Installation

1. Clone or download this repository

2. Install dependencies:

```bash
npm install
```

3. Start the development server, it will open your portfolio in the browser on [Port 3000](http://localhost:3000)

```bash
npm run dev
```
